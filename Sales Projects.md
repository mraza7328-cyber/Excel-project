# Vrinda Store Data Analysis (Excel Project)

## Project Overview

This project analyzes the sales performance of Vrinda Store using Microsoft Excel. The objective is to clean the data, create interactive dashboards, and generate business insights to support decision-making.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Cleaning

## Dataset

The dataset contains sales records including:

- Order ID
- Gender
- Age Group
- State
- Channel
- Product Category
- Sales
- Quantity
- Order Status

## Business Questions

- Which gender contributes the highest sales?
- Which age group purchases the most?
- Which states generate maximum revenue?
- Which sales channels perform best?
- What are the monthly sales trends?

## Key Insights

- Women contribute around 65% of total sales.
- Maharashtra, Karnataka, and Uttar Pradesh are the top-performing states.
- Customers aged 30–49 years contribute the highest sales.
- Amazon, Flipkart, and Myntra are the top sales channels.

## Final Business Recommendation

Target women customers aged 30–49 years in Maharashtra, Karnataka, and Uttar Pradesh through promotional campaigns on Amazon, Flipkart, and Myntra to improve overall sales.
